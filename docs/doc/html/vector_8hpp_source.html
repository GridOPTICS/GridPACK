31 /// A parallel or serial vector of values
32 /**
33  * This class uses the Pimpl idiom for implementation so the
34  * interface is completely free of the underlying library. If
35  * constructed with a parallel environment with only one process, a
36  * serial vector is created, otherwise it's parallel.
37  *
38  * The values in a Vector must be set by the user. Values are not
39  * initialized.
40  *
41  * When a Vector is instantiated it is ready to be filled using calls
42  * to methods like ::set_element. When the Vector is filled, all
43  * processors must be notified that it is ready to use with a call to
44  * ::ready().
45  *
46  *
47  */
